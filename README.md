# ShopSphere – React TypeScript E-Commerce UI

A modern and responsive e-commerce web application built using **React + TypeScript + Tailwind CSS**.
This project demonstrates real-world features like product browsing, cart management, checkout flow, and order confirmation.

## Features

* Home page with categories
* Add to Cart functionality
* Buy Now (direct checkout)
* Cart quantity management
* Persistent cart using localStorage
* Checkout form with validation
* Order success page
* Fully responsive design
* Modern UI with Tailwind CSS

---

## Tech Stack

* React (Vite)
* TypeScript
* Tailwind CSS
* React Router DOM
* Axios (API calls)
* LocalStorage (state persistence)
